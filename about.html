Acerca de esta app

Esta es una app web de ejemplo para tu evaluación. Cumple con:

  • NavBar, Hero (en Inicio) y Footer en todas las páginas.
  • Otra página con información diferente (esta misma).
  • Funcionalidad básica: crear, listar, buscar, editar y borrar notas guardadas en localStorage.
  • Despliegue listo para GitHub Pages (HTML/CSS/JS estático).

¿Cómo se guardan las notas?

Las notas se guardan localmente en tu navegador usando localStorage bajo la clave notas_pro_v1. Si borras los datos del navegador o cambias de dispositivo, las notas no se sincronizan. Para respaldar, exporta a JSON (botón en la página Notas) y guarda el archivo.
